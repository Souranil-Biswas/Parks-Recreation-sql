Parks and Recreation SQL Project

Advanced SQL Analysis on Employee and Salary Data

Project Overview

This SQL project analyzes employee demographics, salary distributions, departmental bonuses, and classification logic within the Parks and Recreation department. Using Common Table Expressions, joins, window functions, and conditional logic, key business insights were extracted.

Data Models

  • employee_demographics: Stores personal details like name, age, gender, and birth date.
  • employee_salary: Contains job roles, salary figures, and department links.
  • parks_departments: Maps department IDs to department names.

Queries and Results
